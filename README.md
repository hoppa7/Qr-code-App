# QR Code Generator

A simple Python GUI app to generate QR codes from text, URLs, or file paths and save them as PNG files. Built with Tkinter, qrcode, and Pillow.

---

## Features

* Generate QR codes from any text or link.
* Preview QR code in the app.
* Save QR code as a PNG file.

---

## Requirements

* Python 3.7+
* Libraries: qrcode, Pillow, customtkinter, tkinter

Install dependencies:

```
pip install qrcode[pil] pillow
pip install customtkinter
pip install qrcode
pip install tkinter
```

---

## Usage

```
python main.py
```
or run main.exe in the QRcode-app directory

1. Enter text or link.
2. Click **Generate QR Code** to preview.
3. Click **Save QR Code** to save as PNG.
